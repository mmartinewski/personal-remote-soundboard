## Personal Soundboard Player v0.8.3

### Dashboard — clip cards

- **Play in… shortcut** — ▶ button next to the clip menu opens the layout area picker directly (video clips with layout areas).
- **Clip type icons** — center overlay shows a **video camera** icon for video clips and the **speaker** icon (Wikimedia Commons) for audio clips.
- **Play destination hint** — video cards show `Play → {area}`; audio cards show `Play → Audio clip`.
- **Cleaner card metadata** — removed the **Video** badge and the redundant **Browser overlay** label next to the category.
- **Full title tooltip** on video clip titles when truncated.

### Upgrade from v0.8.2

Install over v0.8.2. No database or OBS URL changes. Refresh the dashboard after updating if it was already open.
